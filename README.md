# KKKKK ESP32

Firmware `ESP32` per la gestione della parete LED.

## Toolchain

- `VS Code`
- `PlatformIO`
- framework `Arduino` su `ESP32`

## Struttura

- `src/` bootstrap e wiring principale
- `include/` header condivisi
- `lib/` moduli applicativi
- `docs/` riferimento alla documentazione
- `data/` payload e file di test futuri
- `test/` test logici futuri

## Stato Attuale

Questo scaffold contiene:

- bootstrap minimo compilabile;
- macchina a stati runtime base;
- moduli stub per network, API, storage, circuiti e LED;
- base pronta per implementare le API definite nelle specifiche.
